# 🐾 Pet Link – Cuidando a quienes querés

**Pet Link** es una aplicación web pensada para registrar mascotas mediante un sistema de identificación por código QR. Cada mascota tiene su propio perfil, al cual se accede escaneando el código del collar. Así, si se pierde o necesita asistencia, cualquier persona puede encontrar al dueño al instante.

Desarrollado con ❤️ en Corrientes.

---

## 🚀 Características principales

- 📷 **Escaneo de QR** para encontrar información de la mascota
- 📝 **Perfiles de mascotas** con fotos, cuidados especiales y datos del dueño
- 🐶 **Blog “Pet Society”** para compartir historias reales
- 🏠 **Sección de adopciones** con postulación y gestión de entregas
- 🛠️ **Panel de administración** para gestionar dueños, mascotas, y reportes
- 📱 **Diseño responsive** y estética amigable estilo app premium

---

## 🧱 Tecnologías usadas

- ⚛️ **React** – Interfaz moderna y dinámica
- 🎨 **Tailwind CSS** – Estilos rápidos y limpios
- 🐬 **Supabase** – Backend, autenticación y base de datos en tiempo real
- 📦 **Vite** – Build y dev server ultrarrápido
